# smartmet-library-smarttools

Part of [SmartMet Server](https://github.com/fmidev/smartmet-server). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-server) for an overview of the ecosystem.

## Overview

The smarttools library provides scripting support for the SmartMet editor and command-line tools. It contains various tools needed to enable scripting capabilities in SmartMet applications.

## Usage

Used by the SmartMet editor for scripting, and by [smartmet-qdtools](https://github.com/fmidev/smartmet-qdtools) command-line tools.
